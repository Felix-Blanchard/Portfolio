Reinforcement Learning Honours Research Project

I am currently working on a Reinforcement Learning research project for my honours. In this project I am investigating how various reinforcement learning algorithms can be used to post-train LLMs (qwen3) to be better at domain specific judgement tasks (An example being a medical triage operator, or any other high trained specialist). This has involved surveying frontier research and filtering out the useful and relevant work. I am planning on implementing an open source RLTF-FM trainer package for TRL Hugging Face for any future researchers to use in the next few months!

Cricket Spin bowling, ball type classifier

For my computer vision course I am developing a classifier for cricket spin bowling deliveries. Using MediaPipe's pose and hand landmark models alongside OpenCV blob detection, I extract key visual cues from a bowler's run up, action and finally release. These features are then used to incrementally update a Naive Bayes probability distribution, producing a real time estimate of the delivery type before the ball is released.
