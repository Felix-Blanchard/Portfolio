Reinforcement Learning Honours Research Project

I am currently working on a Reinforcement Learning research project for my honours. In this project I am investigating how various reinforcement learning algorithms can be used to post-train LLMs (qwen3) to be better at domain specific judgement tasks (An example being a medical triage operator, or any other high trained specialist). This has involved surveying frontier research and filtering out the useful and relevant work. I am planning on implementing an open source RLTF-FM trainer package for TRL Hugging Face for any future researchers to use in the next few months!

Cricket spin bowling delivery classifier

For a university computer vision paper I am developing a classifier for cricket spin bowling deliveries. Using MediaPipe's pose and hand landmark models alongside OpenCV blob detection, I extract key visual cues from a bowler's run up, action and finally release. These features are then used to incrementally update a Naive Bayes probability distribution, producing a real time estimate of the delivery type before the ball is released.

WiFi & BLE Location Tracking

Through my internship at Allied Telesis I investigated various techniques for location tracking in an urban environment. I prototyped solutions using ESP32-C6 devices, implementing WiFi RSSI trilateration using linear and nonlinear least squares and WiFi RTT via the IEEE Fine Time Measurement (FTM) protocol using ESP-IDF's wifi library. I also extended proprietary network hardware with BLE RSSI scanning support, allowing scan data to be propogated across internal pipelines to collector devices for position estimation of nearby Bluetooth Low Energy devices.
